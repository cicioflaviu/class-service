Class-Service for Glofox

* Run the server by executing "go run main.go" in the main package
* The server is running on port 8085
* Run the tests by navigating to the "test" folder and executing "go test"
* Used GIN to setup the server and GORM for the in-memory database
* Mocked the DB for the tests using GO-Mocked (github.com/selvatico/go-mocket)
